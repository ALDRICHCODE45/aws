# Study Plan — AWS Developer Associate

## Filosofía de estudio

- **Rápido** no significa superficial.
- Cada bloque tiene: teoría corta + práctica + checkpoint.
- Si no lo podés explicar con tus palabras, no está aprendido.

## Ruta sugerida (8 semanas)

### Semana 1 — Fundamentos Global Infrastructure
- Regiones, AZ, Data Centers, Edge Locations
- Shared Responsibility Model
- IAM básico (users, groups, roles, policies)

### Semana 2 — Compute + Networking base
- EC2, Auto Scaling, ELB
- VPC, subnets, routing, SG vs NACL

### Semana 3 — Storage + Databases
- S3 (tiers, lifecycle, versioning, replication)
- DynamoDB (partitions, consistency, indexes)
- RDS basics

### Semana 4 — Serverless core
- Lambda, API Gateway, EventBridge
- SQS, SNS, Step Functions

### Semana 5 — Security avanzado
- KMS, Secrets Manager, Parameter Store
- Cognito, encryption at rest/in transit

### Semana 6 — Observability + CI/CD
- CloudWatch, X-Ray, CloudTrail
- CodeBuild, CodeDeploy, CodePipeline

### Semana 7 — IaC y deployment patterns
- CloudFormation / SAM / CDK
- Blue/Green, Canary, rolling

### Semana 8 — Simulacros + refuerzo
- Preguntas tipo examen
- Repaso por debilidades

## Regla de progreso

No pasás de módulo hasta cumplir:

- [ ] Explicación en 90 segundos del tema
- [ ] 1 ejercicio práctico funcionando
- [ ] 70%+ en mini quiz del módulo
